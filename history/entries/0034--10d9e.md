## Approach

Adds a run-length context: the last byte combined with the length of its current run of identical bytes (capped). Models run continuation vs termination — zero/0xFF runs in binary, repeated punctuation/whitespace in text/source, fill bytes in structured data — which the fixed-order and sparse contexts represent only indirectly. The run length is tracked deterministically at each byte boundary; the context carries its own counter + bit-history StateMap. Release round-trip gate passes.
