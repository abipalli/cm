## Approach

Output-neutral WORK reduction completing the mixer fusion. The previous PR fused the 27 layer-1 dot products; this fuses their weight updates, which had the same shape — 27 separate loops, each re-reading the whole 221-element input vector. The fused version sets up per-mixer (err_k*lr_k folded, and a raw pointer to each selected weight row), then does a single pass over the inputs, loading each mix_in[i] once and applying it to all 27 rows: w_k[i] += (x[i]*err_k*lr_k)>>16. err_k*lr_k is folded once (both factors fit i32, so the product and the shifted result are identical to x*err*lr). The 27 rows live in distinct, non-overlapping, never-resized weight buffers, so the raw pointers stay valid and non-aliasing.
