## Approach

Adds eight gap-bigram context models to the bank (26 -> ... -> now extended): the last byte paired with a single byte at a fixed distance k for k = 6..13. These are lighter, lower-order cousins of the 3-sample strided-sparse models added previously, capturing pairwise correlations at medium distances (delimiter/field spacing, indentation, instruction operands) that the contiguous and 3-sample-stride contexts miss.
The existing bank only had gap bigrams at k = 3,4,5. Each new model carries its own adaptive counter and bit-history StateMap into the layer-1 mixers. All are byte-offset patterns with no corpus-specific content. Release and debug (overflow) round-trip gates pass; every corpus file is unchanged-or-smaller in aggregate.
