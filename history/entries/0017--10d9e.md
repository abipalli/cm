## Approach

Adds a second layer-2 combiner and averages it with the existing one in the logit domain. Both mix the same five layer-1 specialist logits, but select their weight row by a different context: the original keys on the last byte (c1), the new one on the within-byte bit position (c0). Their two output logits are averaged before the squash, then the APM chain proceeds as before. Each combiner trains online on its own prediction (PAQ-style), so the ensemble averages two differently-contextualised views of the same specialists.
A layer-2 mixer is tiny (NL1*256 weights), so memory and the CI footprint are unchanged. Every corpus file improves. Release and debug round-trip gates pass.
