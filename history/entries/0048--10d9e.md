## Approach

Adds a sixteenth layer-1 specialist mixer selected by the character classes (letter / digit / whitespace / other, 2 bits each) of the last four bytes. Like the high-nibble specialist, this groups context into a small set of meaningful coarse 'modes' (in-word, in-number, in-whitespace, punctuation) and lets a full mixer specialize its weighting accordingly. Orthogonal to the byte/match/nest/high-nibble selectors. Each specialist trains online on its own prediction. Release and debug round-trip gates pass; broad improvement across text and source.
