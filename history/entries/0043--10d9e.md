## Approach

Restructures the bit-history state byte. Under the reset-recency rule the minority count is always small (<=3), so the symmetric nibble layout (n0,n1 each 0..15) wastes most of its 256 states on unreachable both-large pairs. The new encoding packs (sign, majority 0..31, minority 0..3), reclaiming those states to represent run lengths up to 31 instead of 15. This gives every context model's StateMap finer resolution on runs, with no extra memory (still a u8 state, 256-entry StateMaps). Release and debug round-trip gates pass; aggregate improves across text/source.
