## Approach

Adds a new context family using the differences between consecutive recent bytes (c1-c2, c2-c3, c3-c4, mod 256) rather than the byte values themselves. Deltas expose gradient/stride patterns in numeric, tabular and executable data that absolute-value contexts blur — translation-invariant structure. General transform, no corpus-specific logic; carries its own counter + bit-history StateMap. Release and debug round-trip gates pass; broad improvement including the binary file.
