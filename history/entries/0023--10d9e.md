## Approach

Adds a sixth layer-1 specialist mixer selected by the current match state — the order-6 match-length bucket (min 15) plus whether the order-10 and order-12 match models are active (64 contexts). The existing layer-1 specialists select on byte/bit-position contexts (last byte+match flags, bit position, order-1/2/3); none lets a full mixer specialize on *how strongly the match models agree*. Because match state is orthogonal to byte context, this specialist learns a distinct full-input weighting for in-match vs out-of-match regions, which the byte-context specialists cannot represent. (The same match-state context already helped as a layer-2 combiner; here it drives a full layer-1 mixer over all 143 inputs.)
Each specialist trains online on its own prediction. The added mixer is tiny (64 rows). Release and debug round-trip gates pass; every corpus file improves.
