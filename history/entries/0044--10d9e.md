## Approach

Adds a new context family using only the high nibble of each byte, ignoring the low-bit (operand/value) noise. Two contexts: the top nibbles of the last 5 bytes, and an order-8-coarse version over the last 8 bytes. This captures coarse structure — x86 opcode classes in the executable, and punctuation/letter-class shape in text/source — that the full-byte contexts blur. General masking, no corpus-specific logic; each carries its own counter + bit-history StateMap. Release and debug round-trip gates pass; improves across the board including the stubborn binary file.
