## Approach

Extends the 2D / byte-above model family with two more contexts that read the upper neighbourhood of the previous line:
- byte above + byte above-right (the character that came next on the line above) — strong when the current line copies/continues the one above.
- a 3-wide horizontal window from the previous line (above-left, above, above-right).
These complement the existing above/above-left/two-rows-up contexts and capture line-to-line copying that the single byte-above misses. Deterministic from decoded bytes; each carries its own counter + StateMap. Release and debug round-trip gates pass; repetitive_nci and source_samba improve most.
