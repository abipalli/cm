## Approach

Expands the layer-2 ensemble from two combiners to four, averaged in the logit domain. Each combiner mixes the same five layer-1 specialist logits but selects its weight row by a different context:
- last byte (c1)
- within-byte bit position (c0)
- match state — order-6 match length bucket plus order-10/order-12 match-active flags
- the second-to-last byte (an order-1-at-distance-2 view)
The match-state combiner is the most valuable addition: its context is orthogonal to the byte/bit-position views, so it captures how much to trust the specialists specifically inside long repeats. Four combiners also let the average use a clean power-of-two shift. Each combiner trains online on its own prediction. Layer-2 mixers are tiny, so memory and the CI footprint are unchanged. Release and debug round-trip gates pass; every corpus file improves.
